# AI Contributors Guide

This project supports AI-assisted contributions.

## 🧠 Supported AI Roles

* Code generation
* Refactoring
* Writing tests
* Documentation updates
* Bug detection

## 📦 Contribution Format

AI-generated contributions MUST:

1. Be explainable
2. Include reasoning
3. Include test coverage (if applicable)

## 📄 Example AI PR Structure

```
Title: feat: optimize text chunking

Description:
- Improved segmentation logic
- Reduced memory usage

AI Notes:
- Model: GPT
- Prompt summary: optimize large text processing
```

## ⚠️ Rules

* No blind-generated code
* Must pass lint + tests
* Human review required
