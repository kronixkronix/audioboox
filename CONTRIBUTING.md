# Contributing Guide

## 🧭 Workflow

1. Fork repo
2. Create branch:

   ```bash
   git checkout -b feature/your-feature
   ```
3. Commit changes
4. Push and create PR

## 📌 Commit Convention

Use:

* feat: new feature
* fix: bug fix
* docs: documentation
* refactor: code improvement

Example:

```
feat: add PDF chapter splitting
```

## 🧪 Testing

* Backend: pytest
* Frontend: jest

## 📏 Code Standards

* Python: PEP8
* JS: ESLint + Prettier

## 🧩 Good First Issues

Look for:

* `good-first-issue`
* `help-wanted`

## 📣 Communication

Use GitHub Issues for discussion.
